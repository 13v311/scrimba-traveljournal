a quiz about the .map() method

1. What does the .map() array method do?
- the '.map()' array method takes a current array and returns a new one based on the inner function it is given. each index of the new array represents the corresponding index of the old array after being modified by the function.


2. What do we usually use '.map()' for in React?
- we would use '.map()' to pull raw data from somewhere that can be placed and converted into an array of JSX elements. those elements can then be displayed on the page.


3. Critical thinking: why is using '.map()' better than just creating the components manually by typing them out?
- we often don't have the data ahead of time when building, so we can't manually type them out
- the code doesn't require additional changes whenevr the data changes
